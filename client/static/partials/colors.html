Colors

Slice colors alow for a color palette range starting from a central color and giving five lighter and darker color options.

  • To use a color as a background place the color name in the class of the object.
  • To use a color for text use the color name with -text (EX: plum-text)
  • To use a lighter or darker color option for backgrounds use lighter-1 through lighter-5 and darker-1 through darker-5
  • to use a lighter or darker color option for text use lighter-text-1 through lighter-text-5 and darker-text-1 through darker-text-5

Type-Face

Slice uses CicleFina as the default type-face which offers a unique slim design.

Two additional options are also available by adding the appropriate class tag to the element

  • bold-text: will offer you CicleGordita a wider version of CicleFina
  • shadow-text: will give you access to CicleShadow, a wider version of CicleGordita with drop shadow. The shadow will be the text-color selected with a white fill
