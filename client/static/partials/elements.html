Navigation Bar

Navigation bar code can be placed at the top of your html page before the main body of your code. the <nav> can have the color modified by using one of the color classes. the logo text and link text can have their colors modified by using the "color"-text class.

Modal

Slice modals can be a very versitile element to add to any application. The below two examples show how the modal can be customized for a simple message display or for use as form submission.

Every modal is customizable in width by applying a row to the modal-wraper <div> and a col class to the child modal <div>

The modal itself is purely CSS based, however the trigger for the modal utilizes a simple javascript function to load and close the modal element. This makes the modal possible to use with many different front-end frameworks

Message Modal html

<div id="firstModal1" class="modal-wrapper row">
	<div id="firstModal2" class="modal blueberry lighter-5 col l3">
		<div>
			<p class="salt-text">Put something in here</p>
			<button class="modal-closebtn" onclick="modalClose('firstModal1')">Close</button>
		</div>
	</div>
</div>

Modal js

<script type="text/javascript">
	var modal = function(id1, id2){
		document.getElementById(id1).style.display='block';
		document.getElementById(id2).style.float='none';
	}
	var modalClose = function(id1){
		document.getElementById(id1).style.display='none';
	}
</script>
